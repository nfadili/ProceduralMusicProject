Procedural music generation project using:
    Python 3.5
    vishnubob's python-midi library

To Run:
    This program depends on this library -> https://github.com/nfadili/python-midi. 
    Downlaod and run it first.
    Once installed, clone this repo and from the project's root directory run -> src/main.py.
    
Available keys:
    'Major'
    'Minor'
    'Harmonic Minor'
    'Neopolitan Major'
    'Falmenco'
    'Gypsy'
